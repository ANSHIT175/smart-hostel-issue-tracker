# Smart Hostel Issue Tracker

Problem Statement: Smart Hostel Issue Tracking System

This is a full-stack web application where students can report hostel issues
(plumbing, electricity, cleanliness, internet, etc.) and track their status.
Admins can assign, manage, and resolve issues with analytics and dashboards.

Features (Planned):
- Student login & issue reporting with images
- Issue status flow: Reported → Assigned → In Progress → Resolved
- Admin panel to manage and assign issues
- Category & priority based filtering
- Dashboard with pending vs resolved stats
